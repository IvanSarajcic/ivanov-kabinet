Хлебови

Комбинаторика и систем линеарних једначина

Текст задатка

  • Сваки мушкарац носи два хлеба, жена један, дете пола хлеба.
  • Укупно је n особа које носе n хлебова.
  • Потребно је наћи број тројки (m, z, d) – мушкарци, жене, деца.
  • Улаз: број n, 1 ≤ n ≤ 109.
  • Излаз: укупан број решења.

Математичка формулација

  • Нека су m, z, d цели и не-негативни.
  • Услов броја особа: m + z + d = n.
  • Услов броја хлебова: 2m + z + 0.5d = n.
  • Множимо други услов са 2: 4m + 2z + d = 2n.
  • Одузимањем првог од другог добијамо 3m + z = n → z = n - 3m.

Последице и формула

  • Из прве једначине и израза за z следи d = 2m (увек паран број).
  • Не-негативност z даје ограничење 0 ≤ m ≤ ⌊n/3⌋.
  • За сваки дозвољени m постоји јединствена тројка.
  • Број решења је број целих m у интервалу, тј. ⌊n/3⌋ + 1.
  • Задатак се своди на једноставну формулу.

Имплементација


long BrojResenja(long n)
{
	return n / 3 + 1; // целобројно дељење
}

Решење ради у O(1) и без проблема за велико n.

Примери

  • n = 5 → 2 решења: (1,2,2) и (0,5,0).
  • n = 12 → 5 решења (m од 0 до 4, d = 2m, z = n - 3m).
  • Број деце прати број мушкараца: d = 2m.
