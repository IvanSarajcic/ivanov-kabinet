# Иванов кабинет - Наставни материјали

Централизовано место за све наставне материјале, лекције и презентације.

## Предмети

### 📚 [Програмске парадигме](programske-paradigme/)
- 22 лекције: од програмских језика до логичке парадигме
- Haskell, Prolog, функционално и логичко програмирање
- Reveal.js презентације за часове
- Примери кода са синтакс highlighting
- **Статус**: ✅ Комплетно

### 💻 [Програмирање 2](programiranje-2/)
- Лекције: Коректност, индуктивно-рекурзивна конструкција
- Појмови: Рекурзија (са паралелним примерима у C++ и Python)
- Reveal.js презентације
- **Статус**: 🚧 У развоју

### 🖥️ [ИКТ 1](ikt-1/)
- 9 лекција: Од области рачунарства до звука и слике у дигиталном свету
- Основе рачунарства и информатике за први разред
- Представљање података, дигитални уређаји
- **Статус**: 🚧 У развоју

## Технологије

- **Лекције**: HTML5 + CSS3
- **Презентације**: [Reveal.js](https://revealjs.com/)
- **Фонтови**: Google Fonts (Noto Sans за ћирилицу)
- **Теме**: Прилагођене тамне теме

## Структура пројекта

```
ivanov-kabinet/
├── programske-paradigme/    ← Програмске парадигме (22 лекције)
│   ├── lekcije/            ← HTML лекције
│   └── prezentacije/       ← Reveal.js презентације
├── programiranje-2/        ← Програмирање 2 (IT смер)
│   ├── lekcije/           ← HTML лекције
│   ├── pojmovi/           ← Појмови (рекурзија...)
│   └── prezentacije/      ← Reveal.js презентације
├── ikt-1/                 ← ИКТ 1 (9 лекција)
│   ├── lekcije/          ← HTML лекције
│   └── prezentacije/     ← Reveal.js презентације
├── _shared/              ← Дељени ресурси
│   ├── css/             ← Стилови (course-styles.css)
│   └── images/          ← Слике
├── _planovi/            ← Годишњи и оперативни планови
├── reveal.js/          ← Presentation framework
└── README.md           ← Овај фајл
```

## Инсталација

```bash
# Клонирајте репозиторијум
git clone https://github.com/IvanSarajcic/ivanov-kabinet.git
cd ivanov-kabinet

# Инсталирајте reveal.js (само први пут)
git clone https://github.com/hakimel/reveal.js.git
cd reveal.js
npm install
```

## Коришћење

1. **Лекције**: Отворите HTML фајлове директно у прегледачу
2. **Презентације**: 
   - Отворите презентацију у прегледачу
   - Стрелице за навигацију
   - `S` за режим говорника
   - `F` за цео екран

## Карактеристике

### 🎨 Стилизовање
- Централизовани CSS (`_shared/css/course-styles.css`)
- Синтакс highlighting за код (C++, Python, Haskell, Prolog)
- Тамна IDE тема за код блокове
- Grid layout за паралелне примере кода

### 🌐 Веб компатибилност
- Сви фајлови slugified (latinica, без ћирилице у именима)
- UTF-8 encoding за садржај
- Responsive design

### 📋 Планови
- Годишњи планови рада (`_planovi/`)
- Детаљни планови по часовима
- Екстракција из Word докумената

## Додавање новог предмета

1. Направите фолдер `novi-predmet/`
2. Копирајте структуру из `_shared/TEMPLATE.html`
3. Креирајте `lekcije/`, `pojmovi/`, `prezentacije/` фолдере
4. Користите `_shared/css/course-styles.css` за стилове
5. Додајте у овај README
