Императивна парадигма

Програмске парадигме

Шта је императивна парадигма?

  • Најстарија програмска парадигма
  • Тесно повезана са хардвером
  • Програм = низ наредби
  • Наредбе мењају податке у меморији

Кључни појмови

  • Стање - вредности променљивих
  • Мутабилно стање - мења се током извршавања
  • Наредба доделе - x = 5;
  • Контрола тока - if, while, for
  • Итерација - понављање кроз петље

Пример императивног кода


#include <iostream>

int main() {
    int x = 3, n = 4, i = 0, stepen = 1;
    while (i < n) {
        stepen *= x;
        i++;
    }
    std::cout << stepen << std::endl;
    return 0;
}

Резултат: 81 (34)

Промена стања програма

x n i stepen
3401
3413
3429
34327
34481

Наредба доделе


i = i + 1;

Ово није математичка једначина!

Значење: Променљивој i додели вредност која је за 1 већа од тренутне вредности i

Итерација

  • Основна техника императивних програма
  • Вредност се мења корак по корак
  • Остварује се помоћу петљи
  • У примеру: stepen се мења од 1 → 3 → 9 → 27 → 81

Бочни ефекти

Операције које мењају стање програма:

  • Оператори ++ и --
  • Функције које мењају глобалне променљиве
  • Улазно-излазне операције (испис, читање)
  • Рад са датотекама, мрежом...

Структурирано програмирање

Скокови у асемблеру


   mov eax, 5
pocetak:
   cmp eax, 0
   je kraj
   dec eax
   jmp pocetak
kraj:
   call print

Петља реализована скоковима (условним и безусловним)

GOTO у раним језицима (BASIC)


10 X = 5
20 IF X = 0 THEN GOTO 50
30 X = X - 1
40 GOTO 20
50 PRINT(X)

Композитна парадигма - секвенцијално извршавање са скоковима

Савремени приступ (без GOTO)


int x = 5;
while (x != 0)
    x--;
Console.WriteLine(x);

Структурирано програмирање - петље уместо скокова

Проблеми са GOTO

  • "Спагети кôд" - тешко пратити ток програма
  • Тешко разумевање и одржавање
  • Склоно грешкама
  • Dijkstra: "GOTO statement considered harmful" (1968)

Решење: Структурне наредбе (if, while, for)

Резиме

  • Императивна парадигма = наредбе + стање
  • Итерација кроз петље
  • Бочни ефекти мењају стање
  • Структурирано програмирање > GOTO
  • Језици: C, C++, Java, Python, C#...
