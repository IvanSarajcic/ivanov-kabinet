CSS Синтакса

Структура CSS правила

CSS правило се састоји од селектора и блока декларација: 

селектор {
    својство: вредност;
    својство: вредност;
}

Компоненте CSS правила

1. Селектор

Селектор указује на HTML елемент који желите да стилизујете.

2. Блок декларација

Блок декларација садржи једну или више декларација одвојених тачка-зарезом (;).

3. Декларација

Свака декларација укључује:

  • Име CSS својства
  • Вредност својства
  • Одвојене су двотачком (:)

4. Витичасте заграде

Блокови декларација су окружени витичастим заградама { }.

Пример

У овом примеру сви <p> елементи ће бити центрирани, са црвеном бојом текста:

p {
  color: red;
  text-align: center;
}

Објашњење примера

  • p је селектор у CSS-у (показује на HTML елемент који желите да стилизујете: <p>)
  • color је својство, а red је вредност својства
  • text-align је својство, а center је вредност својства

Више декларација

Можете додати више CSS декларација у један блок. Свака декларација мора бити одвојена тачка-зарезом: 

h1 {
  color: blue;
  font-size: 24px;
  font-weight: bold;
  margin: 10px 0;
}

Важна правила синтаксе

  1. Својства и вредности су одвојени двотачком :
  2. Свака декларација се завршава тачка-зарезом ;
  3. Блок декларација је окружен витичастим заградама { }
  4. CSS није осетљив на размаке и нове редове (али их користите ради читљивости)
  5. CSS је осетљив на велика/мала слова за неке вредности (нпр. имена класа)

Коментари у CSS-у

Коментари се користе за објашњење кода и игнорисани су од стране прегледача:

/* Ово је коментар */
p {
  color: red; /* Ово је такође коментар */
}
