CSS - Увод

Шта је CSS?

Cascading Style Sheets (CSS) је језик за дефинисање стилова који се користи за описивање презентације и стилизовања документа написаног у језику за означавање као што је HTML или XML (укључујући XML дијалекте као што су SVG, MathML или XHTML). CSS је једна од основних технологија World Wide Web-а, поред HTML-а и JavaScript-а.

Раздвајање садржаја и презентације

CSS је дизајниран да омогући раздвајање садржаја и презентације, укључујући распоред, боје и фонтове. Ово раздвајање може:

  • Побољшати приступачност садржаја - садржај се може писати без бриге о његовој презентацији
  • Пружити већу флексибилност и контролу у специфицирању карактеристика презентације
  • Омогућити дељење форматирања између више веб страница навођењем релевантног CSS-а у посебном .css фајлу, што смањује сложеност и понављање у структурном садржају
  • Омогућити кеширање .css фајла ради побољшања брзине учитавања странице између страница које деле фајл

Различити начини приказа

Раздвајање форматирања и садржаја такође омогућава приказивање исте HTML странице у различитим стиловима за различите методе приказа, као што су:

  • На екрану
  • При штампању
  • Гласом (преко прегледача заснованог на говору или читача екрана)
  • На Брајевим тактилним уређајима

CSS такође има правила за алтернативно форматирање ако се садржају приступа са мобилног уређаја.

Каскадирање

Назив cascading (каскадирање) долази од специфициране шеме приоритета која одређује која декларација се примењује ако више од једне декларације својства одговара одређеном елементу. Ова каскадна шема приоритета је предвидива.

Стандарди и подршка

CSS спецификације одржава World Wide Web Consortium (W3C). Internet media type (MIME type) text/css је регистрован за употребу са CSS-ом преко RFC 2318 (март 1998). W3C управља бесплатном услугом валидације CSS-а за CSS документе.

Поред HTML-а, други језици за означавање подржавају употребу CSS-а укључујући XHTML, обичан XML, SVG и XUL. CSS се такође користи у GTK widget toolkit-у.

Зашто користити CSS?

Главне предности CSS-а:

  1. Раздвајање дизајна од садржаја
  2. Једноставније одржавање веб сајта
  3. Боља приступачност
  4. Различити стилови за различите уређаје
  5. Бржа брзина учитавања страница
