Како додати CSS

Увод

Када прегледач чита stylesheet, он ће форматирати HTML документ у складу са информацијама у stylesheet-у.

Постоје три начина уметања stylesheet-а:

  • Екстерни CSS
  • Интерни CSS
  • Инлајн CSS

1. Екстерни CSS

Са екстерним stylesheet-ом, можете променити изглед целог веб сајта променом само једног фајла!

Свака HTML страница мора укључити референцу на екстерни stylesheet фајл унутар <link> елемента, унутар head секције.

Пример HTML странице

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="mystyle.css">
</head>
<body>

  <h1>Ово је наслов</h1>
  <p>Ово је параграф.</p>

</body>
</html>

Садржај екстерног CSS фајла (mystyle.css)

body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

Важно:

  • Екстерни stylesheet може бити написан у било ком текст едитору
  • Мора бити сачуван са .css екстензијом
  • Екстерни .css фајл НЕ СМЕ садржати HTML тагове
  • Не додавајте размак између вредности својства и јединице:
    ❌ Погрешно: margin-left: 20 px;
    ✅ Исправно: margin-left: 20px;

2. Интерни CSS

Интерни stylesheet се може користити ако једна HTML страница има јединствен стил.

Интерни стил је дефинисан унутар <style> елемента, унутар head секције.

Пример

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: linen;
    }

    h1 {
      color: maroon;
      margin-left: 40px;
    }
  </style>
</head>
<body>

  <h1>Ово је наслов</h1>
  <p>Ово је параграф.</p>

</body>
</html>

3. Инлајн CSS

Инлајн стил се може користити за примену јединственог стила на један елемент.

Да бисте користили инлајн стилове, додајте style атрибут релевантном елементу. style атрибут може садржати било које CSS својство.

Пример

<!DOCTYPE html>
<html>
<body>

  <h1 style="color:blue; text-align:center;">Ово је наслов</h1>
  <p style="color:red;">Ово је параграф.</p>

</body>
</html>

Савет: Инлајн стил губи многе предности stylesheet-а (мешањем садржаја са презентацијом). Користите ову методу штедљиво.

Више Stylesheet-ова

Ако су нека својства дефинисана за исти селектор (елемент) у различитим stylesheet-овима, користиће се вредност из последњег прочитаног stylesheet-а.

Пример 1: Интерни после екстерног

<head>
  <link rel="stylesheet" href="mystyle.css">
  <style>
    h1 {
      color: orange;
    }
  </style>
</head>

Ако је интерни стил дефинисан после линка до екстерног stylesheet-а, <h1> елементи ће бити "orange".

Пример 2: Интерни пре екстерног

<head>
  <style>
    h1 {
      color: orange;
    }
  </style>
  <link rel="stylesheet" href="mystyle.css">
</head>

Међутим, ако је интерни стил дефинисан пре линка до екстерног stylesheet-а, <h1> елементи ће бити боје дефинисане у екстерном фајлу.

Каскадни редослед

Који стил ће се користити када је наведено више од једног стила за HTML елемент?

Сви стилови на страници ће "каскадирати" у нови "виртуелни" stylesheet према следећим правилима, где број један има највиши приоритет:

Приоритети (од највишег до најнижег):

  1. Инлајн стил (унутар HTML елемента)
  2. Екстерни и интерни stylesheet-ови (у head секцији)
  3. Подразумевани стил прегледача

Дакле, инлајн стил има највиши приоритет и ће преписати екстерне и интерне стилове и подразумевана подешавања прегледача.

Резиме

Метод Где се налази Предности Када користити
Екстерни Посебан .css фајл Деље се између страница, лако одржавање За читав сајт
Интерни У <style> тагу Специфичан за једну страницу За јединствене стилове странице
Инлајн У style атрибуту Највиши приоритет Ретко, само за појединачне елементе
