CSS Селектори

Шта су CSS селектори?

CSS селектори се користе да "пронађу" (или изаберу) HTML елементе које желите да стилизујете.

CSS селекторе можемо поделити у пет категорија:

  • Једноставни селектори - бирају елементе на основу имена, id-а, класе
  • Комбинаторски селектори - бирају елементе на основу специфичног односа између њих
  • Псеудо-класа селектори - бирају елементе на основу одређеног стања
  • Псеудо-елементи селектори - бирају и стилизују део елемента
  • Атрибут селектори - бирају елементе на основу атрибута или вредности атрибута

1. Селектор елемента

Селектор елемента бира HTML елементе на основу имена елемента. 

p {
  text-align: center;
  color: red;
}

Овде ће сви <p> елементи на страници бити центрирани, са црвеном бојом текста.

2. ID селектор

ID селектор користи id атрибут HTML елемента да изабере специфичан елемент.

ID елемента је јединствен на страници, тако да се ID селектор користи за избор једног јединственог елемента!

Да бисте изабрали елемент са специфичним ID-ом, напишите тараба (#) знак, а затим ID елемента. 

#para1 {
  text-align: center;
  color: red;
}

CSS правило изнад ће се применити на HTML елемент са id="para1".

Напомена: Име ID-а не може почињати бројем!

3. Класа селектор

Класа селектор бира HTML елементе са специфичним class атрибутом.

Да бисте изабрали елементе са специфичном класом, напишите тачку (.) знак, а затим име класе. 

.center {
  text-align: center;
  color: red;
}

У овом примеру сви HTML елементи са class="center" ће бити црвени и центрирани.

Специфицирање елемента

Можете такође навести да само специфични HTML елементи треба да буду погођени класом:

p.center {
  text-align: center;
  color: red;
}

У овом примеру само <p> елементи са class="center" ће бити црвени и центрирани.

Више класа

HTML елементи могу такође референцирати више од једне класе:

<p class="center large">Овај параграф има две класе.</p>
Напомена: Име класе не може почињати бројем!

4. Универзални селектор

Универзални селектор (*) бира све HTML елементе на страници. 

* {
  text-align: center;
  color: blue;
}

CSS правило изнад ће утицати на сваки HTML елемент на страници.

5. Групни селектор

Групни селектор бира све HTML елементе са истим дефиницијама стила.

Да бисте груписали селекторе, одвојите сваки селектор зарезом. 

h1, h2, p {
  text-align: center;
  color: red;
}

У овом примеру смо груписали селекторе како бисмо минимизовали код.

Табела свих једноставних CSS селектора

Селектор Пример Опис
#id #firstname Бира елемент са id="firstname"
.class .intro Бира све елементе са class="intro"
* * Бира све елементе
element p Бира све <p> елементе
element, element div, p Бира све <div> елементе и све <p> елементе

Резиме

  • p - бира све параграфе
  • #mojId - бира елемент са ID-ом "mojId"
  • .mojaKlasa - бира све елементе са класом "mojaKlasa"
  • * - бира све елементе
  • h1, h2, p - бира све наслове h1, h2 и параграфе
