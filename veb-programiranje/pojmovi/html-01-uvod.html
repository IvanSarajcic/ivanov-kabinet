HTML - Увод

Шта је HTML?

Hypertext Markup Language (HTML) је стандардни језик за означавање докумената који су дизајнирани да буду приказани у веб прегледачу. Он дефинише садржај и структуру веб садржаја. Често му помажу технологије као што су Cascading Style Sheets (CSS) и скрипт језици као што је JavaScript.

Како функционише?

Веб прегледачи примају HTML документе од веб сервера или из локалне меморије и приказују их као мултимедијалне веб странице. HTML описује структуру веб странице семантички и првобитно је укључивао назнаке за њен изглед.

HTML Елементи

HTML елементи су градивни блокови HTML страница. Са HTML конструкцијама, слике и други објекти као што су интерактивне форме могу бити уграђени у приказану страницу. HTML пружа начин за креирање структурираних докумената означавањем структурне семантике за текст као што су:

  • Наслови
  • Параграфи
  • Листе
  • Линкови
  • Цитати
  • Други елементи

HTML Тагови

HTML елементи су означени таговима, који се пишу помоћу угластих заграда. Тагови као што су <img> и <input> директно уносе садржај на страницу. Други тагови као што су <p> и </p> окружују и пружају информације о тексту документа и могу укључивати под-елементе.

Напомена: Прегледачи не приказују HTML тагове, већ их користе да интерпретирају садржај странице.

Скриптовање и Стилизовање

HTML може уградити програме написане у скрипт језику као што је JavaScript, који утиче на понашање и садржај веб страница. Укључивање CSS-а дефинише изглед и распоред садржаја.

World Wide Web Consortium (W3C), бивши одржавалац HTML-а и тренутни одржавалац CSS стандарда, подстиче употребу CSS-а уместо експлицитног презентационог HTML-а од 1997. године.

HTML5

Форма HTML-а позната као HTML5 се користи за приказивање видеа и аудија, примарно користећи <canvas> елемент, заједно са JavaScript-ом.
