HTML Атрибути

Шта су HTML атрибути?

HTML атрибути пружају додатне информације о HTML елементима. Они се увек наводе у отварајућем тагу и обично долазе у паровима име/вредност као: име="вредност".

Пример:
<a href="https://primer.com" target="_blank">Линк</a>
У овом примеру, href и target су атрибути.

Глобални Атрибути

Глобални атрибути су атрибути који се могу користити са свим HTML елементима. Испод су наведени најважнији и најчешће коришћени:

Атрибут Таг на који се примењује Опис
class Сви елементи Наводи једно или више имена класа за елемент (референца на класу у стилу)
id Сви елементи Наводи јединствени ID за елемент
style Сви елементи Наводи инлајн CSS стил за елемент
title Сви елементи Наводи додатне информације о елементу
data-* Сви елементи Користи се за чување прилагођених података приватних за страницу или апликацију
hidden Сви елементи Наводи да елемент још увек није или више није релевантан
lang Сви елементи Наводи језик садржаја елемента
dir Сви елементи Наводи смер текста за садржај у елементу
contenteditable Сви елементи Наводи да ли је садржај елемента измењив или не
draggable Сви елементи Наводи да ли се елемент може превући или не

Специфични Атрибути за Честе Елементе

Атрибути за линкове и слике

Атрибут Таг на који се примењује Опис
href <a>, <link> Наводи URL странице на коју води линк
target <a> Наводи где да се отвори повезани документ (_blank, _self, _parent, _top)
src <img>, <script>, <iframe> Наводи URL слике/скрипте
alt <img>, <input> Наводи алтернативни текст када оригинални елемент не успе да се прикаже
width <img>, <video>, <canvas> Наводи ширину елемента
height <img>, <video>, <canvas> Наводи висину елемента

Атрибути за форме

Атрибут Таг на који се примењује Опис
action <form> Наводи где да се пошаљу подаци форме када се форма проследи
method <form> Наводи HTTP метод за слање података (GET или POST)
type <input>, <button> Наводи тип елемента
name <input>, <select>, <textarea> Наводи име елемента
value <input>, <button> Наводи вредност елемента
placeholder <input>, <textarea> Наводи кратак савет који описује очекивану вредност поља
required <input>, <select>, <textarea> Наводи да елемент мора бити попуњен пре слања форме
disabled <input>, <button>, <select> Наводи да елемент треба бити онемогућен
checked <input> Наводи да елемент треба бити унапред одабран (за checkbox или radio)

Атрибути за табеле

Атрибут Таг на који се примењује Опис
colspan <td>, <th> Наводи број колона које ћелија табеле треба да обухвати
rowspan <td>, <th> Наводи број редова које ћелија табеле треба да обухвати

Примери употребе

Глобални атрибути:
<div id="kontejner" class="glavni" style="color: blue;" data-info="важно">Садржај</div>
Линк са атрибутима:
<a href="https://example.com" target="_blank" title="Посети пример">Кликни овде</a>
Слика са атрибутима:
<img src="slika.jpg" alt="Опис слике" width="300" height="200">
Форма елемент:
<input type="text" name="korisnik" placeholder="Унеси име" required>
