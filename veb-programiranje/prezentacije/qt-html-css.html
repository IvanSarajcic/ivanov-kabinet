Категорије подршке

  • Основно форматирање текста ✔
  • Листе, табеле, линкови и слике ✔ (са ограничењима)

Основно форматирање

Таг Статус Напомена
<b>✔Подебљани текст
<i>✔Курзив
<u>✔Подвучено
<s> / <strike>✔Прецртано
<sup> / <sub>✔Индекси
<br>, <hr>✔Прелом реда / линија

Блокови и фонтови

  • <p>, <div>, <center> функционишу (без CSS layout-а).
  • <font> ради, али се не препоручује.
  • <span style="..."> – подршка за color, background-color, величину и стил фонта.
  • <pre> чува размак

Листе

  • <ul>, <ol>, <li> – ✔потпуно подржани.
  • <dl>, <dt>, <dd> – ✘нису подржани.
  • Нема напредних стилова нумерације.

Табеле

  • <table>, <tr>, <td>, <th> – ради основно рендеровање.
  • <thead>, <tbody>, <tfoot> се игноришу, али не сметају.
  • CSS особине попут width, border-collapse немају ефекат.

Линкови и слике

  • <a href> – ради, потребно setOpenExternalLinks(true).
  • target атрибути се игноришу.
  • <img src> – подржава локалне путање, ресурсе и URL-ове.
  • width, height раде; style је ограничен.

Шта није подржано

  • HTML5 тагови попут <video>, <audio>, <canvas>.
  • Структурни елементи: <article>, <header>, <nav>…
  • Формуларски елементи: <form>, <input>, <button>.
  • Скриптови, уграђени стилови, <iframe>, <embed>.

CSS – ограничења

  • Подржани inline стилови: боје, фонтови, text-decoration, делимично vertical-align.
  • margin, padding, border – делимично и непоуздано.
  • Нема подршке за display, position, flexbox, grid, анимације.
  • Препорука: држати се минималистичког стилизовања.
